

# 📊 **Customer Segmentation using Clustering**

## 🧾 **1. Project Title**

Customer Segmentation using Unsupervised Machine Learning Techniques

---

## ❗ **2. Problem Statement**

Businesses often struggle to understand customer behavior and target the right audience effectively. Without proper segmentation, marketing strategies become inefficient and customer retention decreases.
This project aims to identify meaningful customer segments using clustering algorithms to enable data-driven decision-making and targeted marketing.

---

## 📁 **3. Dataset Description**

The dataset used in this project contains customer-related information such as:

* Age
* Annual Income
* Spending Score

The dataset is clean and structured, making it suitable for applying clustering techniques to analyze customer purchasing behavior.

---

## 🤖 **4. Algorithms Used**

### 🔹 K-Means Clustering

* Partitions data into K clusters
* Works based on distance from centroids
* Best suited for well-separated data

### 🔹 DBSCAN (Density-Based Spatial Clustering)

* Groups data based on density
* Can detect noise and outliers
* Does not require number of clusters in advance

### 🔹 Hierarchical Clustering

* Builds clusters in a tree-like structure
* Can be visualized using dendrogram
* Useful for understanding cluster relationships

## 📊 **6. Key Results**

* **Number of Clusters Found:** 4
* **Best Algorithm:** K-Means Clustering
* **Customer Segments Identified:**

  * High-value customers
  * Budget customers
  * Occasional buyers
  * At-risk customers

### 💡 **Business Insights**

* High-value customers contribute most to revenue
* Budget customers respond well to discounts
* At-risk customers need retention strategies
* Segmentation enables targeted marketing
